  • 一直想自己集成一个H5单页滑动插件，mSwipe.js就这样产生了。

    写mSwipe.js的时候，查看过其他滑动效果，研究了他们的源码，其中主要Swipe.js和Iscroll.js

  • container为切换页面的包裹容器，是一个必填参数，可以为一个ID字符串或者DOM对象，

    options为配置参数，可配置参数见下表。

    参数 说明 默认值
    selector 目标切换页面的选择器 .mSwipe-items
    animated 动画方式，目前主要有三种：0移动，1缩放，2覆盖 0
    scaleTo 缩放系数，当动画方式为1时生效，缩放系数在[0,1)内有效 0.2
    startSlide 默认开始页面，便于调试指定页面 0
    speed 页面切换动画持续时间，单位毫秒 400
    distanceValue 切换临界值，只有当触屏超过临界值，才会切换下一页 100
    direction 切换方向，x水平和y垂直,传递xy之外的字符都取默认y y
    disableToNext 是否禁止向上（左）滑动 false
    disableToPrev 是否禁止向下（右）滑动 false
    disableToLoop 是否开启循环切换 false
    disablePreventDefault 是否禁止默认事件 true
    disableStopPropagation 是否禁止事件冒泡 true
    callback 回调函数，可以带mSwipe的实例对象参数 null
    bindToWrapper 事件是否绑定在容器元素上 true
